# Refresh Knowledge Graph

## Objective
Refresh the repository knowledge graph and structural index.

## Implementation Steps
- Run the `graphify update .` command via the `v12-graphifier` agent (or via `run_shell_command`) to update the index.

## Verification
- Verify that `graphify-out/graph.json` and `graphify-out/GRAPH_REPORT.md` are updated with the latest repository structure.
