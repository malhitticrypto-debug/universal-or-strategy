## Quality Gate Breakdown

### Blocking Checks (8)
1. **ASCII-Only** - Zero non-ASCII characters
2. **Build** - Zero compilation errors
3. **Unit Tests** - 100% pass rate
4. **Lint** - Zero Roslyn violations
5. **Formatting** - Zero CSharpier issues
6. **PR Hygiene** - Diff <10k characters
7. **Complexity** - CYC ≤ 15 (Jane Street aligned)
8. **Security** - Zero secrets (Gitleaks + Snyk) [WARNING initially]

### Warning Checks (5)
9. **Markdown Links** - Zero broken links
10. **Dead Code** - Zero unreachable methods
11. **Codacy Preview** - Zero errors (API-based)
12. **Semgrep** - Zero security findings
13. **CodeRabbit AI** - Zero critical/high issues [WARNING until 2026-06-09]

## CodeRabbit Integration Details

### Installation
```bash
# Option 1: Install script
curl -fsSL https://cli.coderabbit.ai/install.sh | sh

# Option 2: Homebrew
brew install coderabbit
```

### Authentication
```bash
# Browser-based (local development)
cr auth login

# API key (CI/CD)
cr auth login --api-key "$CODERABBIT_API_KEY"
```

### Review Modes
- `--agent`: Structured JSON for automation (used in pre_push_validation.ps1)
- `--plain`: Human-readable terminal output
- `--interactive`: Terminal UI for manual review

### Pricing
- **Free tier**: Limited daily usage, basic static analysis
- **Paid plans**: Learnings-powered reviews, higher rate limits
- **Usage-based Add-on**: $0.25/file, unlimited reviews

### Validation Period
- **2026-05-26 to 2026-06-09**: WARNING mode (non-blocking)
- **After 2026-06-09**: BLOCKING mode (must pass to push)

## Integration Points
